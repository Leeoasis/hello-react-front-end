## 💻 Getting Started <a name="getting-started"></a>

to run on your localhost clone the project to local folder

### Setup

> - git clone https://github.com/Leeoasis/hello-react-front-end.git
> - cd hello-react-front-end
> - npm install

### Prerequisites

In order to run this project you only need:

- latest web-browser
- node.js and npm

### Usage

To run the project, execute the following command:
first run the rails API server by using the following command:

```sh
 git clone https://github.com/Leeoasis/hello-rails-back-end.git
 cd hello-react-back-end
 bundle install
 rails s
 http://127.0.0.1:3000
```

To run front-end use the following command:

```sh
 cd hello-react-front-end
 npm start
 http://localhost:3001/ #opened automatically
```
